2. The Page Breakdown (The "All Pages" Part)

Listing every page is a ubiquitous requirement for case studies. It makes the logic clear:

index.php: The landing page showing the top-rated films.

list.php: The full library with a search bar to filter by title or director.

detail.php: Shows specific movie info and approved reviews.

review.php: The form where logged-in users submit their ratings.

admin.php: The dashboard for site statistics and management links.

manage-reviews.php: Where the admin approves or deletes pending reviews.

manage-users.php: For managing the community and deleting accounts.
